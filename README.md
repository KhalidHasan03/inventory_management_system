# 📦 Inventory Pro Management

A modern **Inventory Management & Point of Sale (POS)** system built with **Laravel 10**, designed to simplify inventory tracking, sales processing, customer management, employee payroll, and business reporting—all from a single dashboard.

Built with **Laravel**, **Tailwind CSS**, **Alpine.js**, **Chart.js**, and **MySQL**, the application offers a fast, responsive, and user-friendly interface suitable for retail stores, wholesalers, and small to medium-sized businesses.

---

## ✨ Key Features

- 📦 Product, Category & Supplier Management
- 🛒 Powerful Point of Sale (POS)
- 👥 Customer & Employee Management
- 💰 Expense & Salary Tracking
- 📊 Dashboard with Business Analytics
- 📈 Sales & Profit/Loss Reports
- 📄 Excel Export Support
- 🔍 Global AJAX Search
- 🔐 Authentication & Role-Based Access Control
- 🌙 Modern Dark UI with Responsive Design

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/inventory-pro-management.git

cd inventory-pro-management

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

php artisan storage:link

npm run build

php artisan serve
```
